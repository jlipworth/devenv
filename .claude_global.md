## CLI Tool Preferences (Rust alternatives)
When using Bash, prefer these Rust-based CLI tools over their traditional counterparts:
- Use `fd` instead of `find` for file searching
- Use `eza` instead of `ls` for directory listings
- Use `rg` (ripgrep) instead of `grep` for text searching
- Use `bat` instead of `cat` for file display
- Use `dust` instead of `du` for disk usage analysis
- Use `sd` instead of `sed` for find & replace operations
- Use `procs` instead of `ps` for process listing
- Use `z` (zoxide) instead of `cd` for directory navigation
